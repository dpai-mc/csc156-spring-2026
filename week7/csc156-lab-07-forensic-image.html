Lab 07: Forensic Disk Image

Downloading, extracting, and preparing the 4orensics evidence image

💾

This week's lab uses a raw forensic disk image named 4orensics.001. The .001 extension indicates a standard raw (dd-style) image format, which is natively supported by Autopsy, FTK Imager, and most other forensic analysis tools used in this course.

⬇ Download the Image

The image is distributed as a .7z compressed archive to keep the download manageable. You will need to extract the archive before you can open it in your forensic tools.

📦 Extracting the Archive with NanaZip

To extract the .7z archive you need a tool that supports the 7-Zip format. The recommended tool for this course is NanaZip — a modern, open-source file archiver that handles .7z, .zip, .tar, and dozens of other formats.

NanaZip can be installed from three official or reputable sources. Choose whichever method suits your setup:

Install NanaZip

  1. Microsoft Store (recommended on Windows 10/11)

    • Open the Microsoft Store app.
    • Search for NanaZip.
    • Install the one published by M2Team / Kenji Mouri.

  2. Official GitHub Releases (MSIX packages & previews)

    • Go to the NanaZip GitHub releases page.
    • Download the latest stable .msixbundle (or preview if you want the bleeding edge).
    • Double-click the downloaded file to install via App Installer.

  3. Windows Package Manager (winget)

    • Open an elevated PowerShell or Terminal window.
    • Run the following command:
    PowerShell winget install -e --id M2Team.NanaZip
💡

Already have 7-Zip? Standard 7-Zip also handles .7z archives and works fine for this lab. NanaZip is simply the instructor's current preference for its cleaner Windows integration and active development.

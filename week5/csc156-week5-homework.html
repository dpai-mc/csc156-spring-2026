Background Context: Week 5 Recap

Throughout Week 5, you've gained in-depth knowledge of several critical file systems: FAT (Module 5.2), NTFS (Module 5.3), and Linux/macOS file systems (ext4, HFS+, APFS) (Module 5.4). You've also learned about deleted file recovery and carving techniques (Module 5.5). This homework challenges you to synthesize this information, comparing their structures, forensic artifacts, and recovery potential to deepen your analytical skills.

    • Max Volume Size (approx.):

Part B: Forensic Scenario Analysis (8 points)

Analyze the following two forensic scenarios. For each scenario, describe the most likely file system(s) involved, the expected recovery challenges, and your recommended approach for deleted file recovery, justifying your choices with concepts from Week 5.

Scenario 1: The Old USB Drive

An old 16GB USB flash drive was found in a suspect's drawer. It was last used on a Windows XP machine. The suspect claims it's empty, but a quick check shows it was recently "quick formatted." You suspect important documents were deleted from it.

  • Identify: What file system(s) are most likely to be present on this drive? Justify your answer.
  • Challenges: What specific challenges would you anticipate for deleted file recovery given the file system type, the quick format, and the age of the OS?
  • Approach: Outline your primary and secondary approaches for recovering deleted files. Explain why you would choose these methods (e.g., metadata-based, file carving) and what specific tools (e.g., fls, Scalpel) you would use, referencing their strengths and weaknesses.

Scenario 2: The Modern Laptop

A modern MacBook Pro (running macOS Ventura) is seized from a suspect. The primary internal drive is an SSD. The suspect claims they deleted all incriminating evidence hours before seizure. You need to recover any deleted files, especially documents and images.

  • Identify: What file system(s) are most likely on the internal SSD?
  • Challenges: What are the most significant challenges for deleted file recovery in this scenario, considering the storage type (SSD) and operating system? How does TRIM impact your expectations?
  • Approach: Describe your recommended approach for attempting deleted file recovery. Discuss the likelihood of success for metadata-based recovery versus file carving. What specific features of the macOS file system (e.g., snapshots, encryption) might offer opportunities or additional challenges?

Part C: Critical Analysis (4 points)

Reflect on the evolution of file systems and storage technologies discussed this week.

Tasks:

  1. Identify what you believe is the **single most impactful challenge** that modern file systems and storage technologies (e.g., SSDs, TRIM, native encryption, complex metadata structures) pose to digital forensic investigations today.
  2. Explain **why** this challenge is so significant, drawing on specific examples from the Week 5 modules.
  3. Propose **two concrete strategies or countermeasures** that forensic examiners can employ to mitigate this challenge in their investigations.

✨ Week 5: Overview

Week 5 builds directly on Week 4's hardware and storage foundation by exploring how operating systems logically organize data on physical storage devices. This week introduces file system architectures, metadata structures, and the critical forensic techniques of deleted file recovery and file carving.

Understanding file systems is the cornerstone of digital forensics. Every piece of digital evidence resides within a file system, and a forensic examiner's ability to locate, preserve, and recover this evidence hinges on a deep understanding of these structures.

🎓
CSLO Alignment: This week directly addresses **CSLO 2: Conduct thorough examinations of computer hard disk drives, and other electronic data storage media to recover information and electronic data.** You will learn the fundamental mechanisms by which digital evidence is stored and retrieved.

📖 Week 5 Learning Modules

This week consists of **six comprehensive learning modules** that progress from foundational file system concepts to advanced recovery techniques. Each module builds upon the previous, creating a complete understanding of how operating systems organize data and how forensic examiners can leverage this knowledge to recover evidence.

Module Breakdown:

5.1 File System Fundamentals ▼
⏰
Estimated Time: 50 minutes
Description: Introduction to what file systems are, their purpose, core components (boot sector, allocation structures, metadata, data area), and their overall forensic relevance.
5.2 FAT File System Deep Dive ▼
⏰
Estimated Time: 55 minutes
Description: Detailed examination of the FAT12, FAT16, and FAT32 file systems, including their structures, metadata, deletion process, and forensic recovery considerations.
5.3 NTFS Deep Dive ▼
⏰
Estimated Time: 70 minutes
Description: In-depth study of the NTFS file system, focusing on the Master File Table (MFT), attributes, journaling, Alternate Data Streams (ADS), and advanced forensic analysis techniques.
5.4 Linux & macOS File Systems ▼
⏰
Estimated Time: 50 minutes
Description: Overview of ext2/ext3/ext4 for Linux and HFS+/APFS for macOS, including their unique structures (inodes, snapshots), timestamps, and specific forensic challenges.
5.5 Deleted File Recovery & File Carving ▼
⏰
Estimated Time: 55-60 minutes
Description: Covers the critical techniques of metadata-based deleted file recovery and file carving, including file signatures, tool configuration, and challenges posed by SSDs and TRIM.
5.6 Practical Application & Lab Preparation ▼
⏰
Estimated Time: 40-45 minutes
Description: Hands-on preparation for Lab Activity 5, focusing on using Autopsy, The Sleuth Kit (TSK), and carving tools to analyze file systems and recover deleted data.

📝 Week 5 Assignments

This week features one lab activity, one discussion post, and one homework assignment, totaling 43 points. These assignments will allow you to apply the concepts learned in the modules and develop practical forensic skills.

Required Assignments:

📋
Lab Activity 5: Deleted File Recovery and File System Analysis

Points: 20 points
Description: Develop practical skills in analyzing file systems, recovering deleted files, and interpreting file system metadata for forensic investigations. This lab involves identifying file systems, performing metadata-based recovery on FAT32, NTFS, and ext4 (if applicable), and conducting file carving.

💬
Discussion Post 5: File System Forensic Challenges

Points: 3 points (Initial Post: 2 pts, Responses: 1 pt)
Description: Reflect on the unique forensic challenges posed by different file systems, the impact of SSDs and TRIM on recovery, and the strategic choice between metadata-based recovery and file carving.

✍️
Homework 5: File System Comparison Analysis

Points: 20 points
Description: Create a detailed comparison of major file systems, analyze forensic scenarios, and critically evaluate the impact of modern storage technologies on investigations.
